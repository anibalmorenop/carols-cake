# Los Dulcitos de Caro

Pagina web en Next.js para la marca de postres caseros **Los Dulcitos de Caro**.

## Scripts

```bash
npm run dev
npm run build
npm run start
```

## Estructura

- `src/app`: rutas y estilos globales de Next.js.
- `src/components`: secciones visuales de la pagina.
- `src/data/products.js`: catalogo mostrado en la galeria.
- `public/img/logo`: logo de marca.
- `public/img/publicaciones`: imagenes de productos usadas en la web.
- `brand`: guia y recursos de marca.

## Despliegue en Vercel

Vercel detecta Next.js automaticamente.

- Framework preset: `Next.js`.
- Build command: `npm run build`.
- Output directory: dejar el valor por defecto.
